# VK Multi Tool

## Описание

VK Multi Tool — это скрипт на Python для автоматизации взаимодействия с API ВКонтакте. Он позволяет искать пользователей, добавлять их в друзья, отправлять сообщения и обрабатывать посты друзей. Скрипт использует библиотеку `vk_api` для работы с API ВКонтакте и поддерживает ведение статистики о выполненных действиях.

## Функционал

- Поиск пользователей по различным критериям (возраст, пол, город и т.д.)
- Добавление пользователей в друзья
- Отправка сообщений новым друзьям
- Лайки и комментарии к постам друзей
- Сохранение и загрузка статистики в формате JSON
- Логирование действий и ошибок в отдельные файлы

## Установка

1. Убедитесь, что у вас установлен Python 3.6 или выше.
2. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/ваш_пользователь/vkmultitool.git
   cd vkmultitool
   ```

3. Установите необходимые зависимости:

   ```bash
   pip install vk_api tqdm
   ```

4. Создайте файл `config.json` в папке аккаунта с вашим токеном доступа:

   ```json
   {
       "access_token": "ВАШ_ТОКЕН_ДОСТУПА"
   }
   ```

## Использование

1. Запустите скрипт:

   ```bash
   python vkmultitool.py
   ```

2. Следуйте инструкциям на экране для выбора аккаунта и выполнения действий.

## Логирование

Все действия и ошибки будут записываться в файлы `vk_unified_script.log` и `error.log` соответственно.

Описание работы скрипта

Скрипт vkmultitool.py предназначен для автоматизации различных действий во ВКонтакте с использованием API. Он выполняет следующие основные функции:

1. Авторизация
Скрипт загружает токен доступа из файла config.json, который должен находиться в папке аккаунта.
Если токен недействителен или истёк, пользователю предлагается обновить его, перейдя по ссылке для авторизации.
2. Выбор действий
После успешной авторизации пользователю предлагается выбрать одно из следующих действий:

Поиск пользователей и сохранение их ID в файл:

Пользователь может задать параметры поиска (пол, возраст, город и т.д.), а также выбрать тематики групп для фильтрации.
Найденные пользователи будут сохранены в файл user_ids.txt.
Добавление друзей из файла:

Скрипт загружает ID пользователей из файла user_ids.txt и отправляет запросы на добавление в друзья с заданной задержкой.
Отправка сообщений новым друзьям:

Скрипт отправляет сообщения новым друзьям с упоминанием их имени и комплиментом.
Обработка постов друзей:

Скрипт может ставить лайки и/или оставлять комментарии на постах друзей.
Обновление токена:

Пользователь может обновить токен доступа, если он истёк.
Отправка сообщений конкретным пользователям:

Пользователь может ввести ID пользователей вручную или загрузить их из файла, после чего отправить им сообщения.
3. Статистика
Скрипт ведет учет различных действий, таких как количество найденных пользователей, добавленных в друзья, отправленных сообщений и т.д.
Статистика сохраняется в формате JSON и может быть загружена при следующем запуске.
4. Логирование
Все действия и ошибки записываются в лог-файлы: vk_unified_script.log для обычных действий и error.log для ошибок.
Это позволяет отслеживать работу скрипта и выявлять возможные проблемы.
